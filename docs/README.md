# 文档导航

这个目录存放开源版的使用说明、部署说明和架构文档。

This directory contains the main usage guides, deployment notes, and architecture documents for the open-source edition.

## 建议阅读顺序

1. [快速开始](quick-start.md)
   适合第一次启动项目、登录系统和确认默认演示数据。
2. [配置说明](configuration.md)
   适合补充 AI、Sonic、外部系统和安全相关配置。
3. [架构文档](architecture.md)
   适合理解前后端、执行器、worker 和主要扩展点。
4. [生产部署](deployment.md)
   适合部署到服务器、共享环境或准备正式落地。
5. [页面截图说明](images/README.md)
   适合维护 README 和 quick start 中使用的页面截图。

## 按场景查找

- 想先跑起来：看 [快速开始](quick-start.md)
- 想知道变量怎么配：看 [配置说明](configuration.md)
- 想理解系统结构：看 [架构文档](architecture.md)
- 想部署到正式环境：看 [生产部署](deployment.md)
- 想更新截图或文档素材：看 [页面截图说明](images/README.md)

## For English Readers

- Start with [quick-start.md](quick-start.md) if you want to run the platform locally first
- Read [configuration.md](configuration.md) for environment variables and integration settings
- Read [architecture.md](architecture.md) for the system structure and extension points
- Read [deployment.md](deployment.md) before deploying to a server or shared environment
- The configuration and deployment documents now include short English notes for the main operational points

## 补充说明

- 开源版默认只提供最小可运行能力和通用演示数据，不包含真实业务资产。
- 所有第三方集成默认保持可选接入，不是本地启动前置条件。
- 如果页面、功能或配置发生明显变化，建议同步更新对应文档和截图。
