# 页面截图说明

本目录存放开源版 README 使用的真实页面截图。

当前文件：

- `login-page.png`：登录页
- `requirements-page.png`：需求列表
- `cases-page.png`：用例库
- `dashboard-page.png`：质量看板
- `settings-page.png`：系统设置

维护建议：

- 优先使用开源版当前实际页面截图，不使用带私有数据的业务页面
- 截图前确认页面中不包含真实账号、真实需求标题、内网地址或敏感配置
- 如需替换截图，尽量保持文件名不变，这样 README 不需要同步改链接
- 若页面视觉变化较大，建议同时更新 README 中的截图顺序和说明文字
