# 开源发行审计

本发行目录由一个不含 Git 历史的源码快照生成，并完成以下处理：

当前开源版已同步功能基线，并重新执行完整脱敏扫描、后端测试和前端构建。

- 删除本地环境文件、数据库、上传文件、截图、登录态、安装包和构建产物。
- 删除内部 Git 仓库初始化脚本和私有仓库地址。
- 将内网 IP、组织域名、人员姓名、业务端名称和 APK 示例替换为通用配置。
- 外部 SSO、任务系统、飞书、Sonic 与自动化框架均改为默认关闭、显式配置。
- 管理员密码、数据库密码和 JWT 密钥由部署者提供。
- 使用 `.example.test` 和 RFC 5737 地址作为不可路由示例。

发布前仍应运行 `scripts/check-secrets.ps1`。脚本会检查密钥、内网地址、绝对路径、业务残留和 Git 作者邮箱；正式发布时仍建议再使用独立密钥扫描工具复核。
